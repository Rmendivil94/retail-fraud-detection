# retail-fraud-detection
End-to-end machine learning pipeline for detecting fraudulent transactions.

Content
- 01_EDA_and_Data_Cleaning.ipynb: Limpieza de nulos, optimización de memoria y análisis exploratorio de variables críticas (Tiempo y Dinero).
- 02_Feature_Engineering_and_Selection.ipynb: Filtros de multicolinealidad, codificación de variables categóricas y Feature Importance.
